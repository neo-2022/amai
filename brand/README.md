# Brand

Этот каталог удерживает канонический branding contour проекта `Art-memory-agent-index (Amai)`.

Источник:
- branding pack был дан пользователем напрямую из `/home/art/Загрузки/Amai label`;
- поэтому эти assets разрешены для baseline именно отдельным прямым приказом пользователя.

## Что здесь лежит

- `amai_lockup.svg`
  - основной lockup для `README`, docs и landing-like поверхностей;
- `amai_mark.svg`
  - квадратный знак для компактного использования;
- `favicon.ico`
  - готовый favicon / app icon baseline;
- `amai_brand_spec.md`
  - текстовая brand-спецификация, импортированная в проект.

## Как использовать

- в `README` и документации использовать `amai_lockup.svg`;
- для favicon, tab icon, dock/app icon использовать `amai_mark.svg` или `favicon.ico`;
- не добавлять градиенты, тени и декоративные усложнения поверх базового знака;
- держать brand-pack как отдельный contour, не смешивая его с runtime/config/code.
