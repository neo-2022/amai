# Amai — Indexed A brand concept

## Core idea
A compact monogram that reads as **A** for **Amai** and also as a vertical **index lane** inside the mark.

## Meaning
- **A** — Amai / Agent
- **Vertical accent** — index, active project, provenance anchor
- **Rounded square** — isolated workspace / namespace boundary
- **Simple geometry** — infrastructure-grade, precise, professional

## Why it works as a favicon
- one dominant silhouette
- high contrast
- no fragile micro-details
- remains readable at 16×16 and 32×32

## Palette
- Midnight Graphite — `#0B1020`
- Cloud White — `#F4F7FB`
- Signal Teal — `#2DD4BF`
- Ink Blue — `#122033`

## Usage
- favicon / browser tab / dock / app icon: use the square mark only
- landing page / README / docs: use the lockup
- avoid gradients, shadows, and extra detail
- keep enough padding around the mark

## Files
- `amai_mark.svg` — main square mark
- `favicon.ico` — ready favicon
- `amai_lockup.svg` — icon + wordmark

Примечание для текущего baseline:
- в этот project contour сейчас импортированы только реально переданные пользователем файлы;
